# User `bin` directories

`~/bin`
:   Standalone files (not symlinks) that we wrote, which aren't significant
    enough to need a project folder in ~/docs/code, but which we'd like on the
    PATH and checking into our personal git dotfiles repo.

`~/.local/bin`
:   3rd party downloaded binaries, or personal projects from ~/docs/code,
    'installed' by copying or symlinking here on the PATH. We don't
    deeply care if we lose this directory, we can download or install again.
